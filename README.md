## About
Backend application for cats searching

- Adding a new cat
- List all cats
- Get a cat with id
- Delete a cat
- Search for a cat by name
- Paginate and sort cat list

## Getting started

Add MONGODB_URI to .env
```shell
cp .env.sample .env
```
Install dependencies and run application:
```
npm install
npm run dev
```
